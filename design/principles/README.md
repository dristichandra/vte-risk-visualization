# Evidence-Based Design Principles

## Principle 1: Counting over Calculating
- **Proposed**: Quantile dots
- **Replaces**: Error bars
- **Evidence**: 1.4x better comprehension (Padilla 2023)
- **Rationale**: 50% of U.S. adults cannot perform basic probability calculations

## Principle 2: Temporal Narrative over Static
- **Proposed**: Dynamic risk trajectories
- **Replaces**: Point-in-time snapshots
- **Evidence**: 28% improvement in early detection (Kia 2021)
- **Rationale**: Static snapshots miss risk evolution across treatment phases

## Principle 3: Transparency over Black-Box
- **Proposed**: Explainable contributing factors
- **Replaces**: Unexplained algorithm outputs
- **Evidence**: 67% override rate for unexplained algorithms
- **Rationale**: Clinicians need to understand why, not just what

## Principle 4: Passive Integration over Alerts
- **Proposed**: Workflow-embedded displays
- **Replaces**: Interruptive pop-up alerts
- **Evidence**: 12% workflow error increase from interruptive systems
- **Rationale**: Alert fatigue leads to 49-96% override rates
